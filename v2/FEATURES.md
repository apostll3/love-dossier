## 💾 Data Management

### LocalStorage

#### Data Structure
```json
{
  "language": "en",
  "theme": "dark",
  "colorTheme": "purple",
  "typesOrder": "[...]",
  "type_communication-log": "{...}",
  "lastVisit": "1234567890",
  "lastBackup": "1234567890",
  "appPassword": "hashed"
}
```

#### Storage Limits
- **Chrome:** ~10 MB
- **Firefox:** ~10 MB
- **Safari:** ~5 MB
- **Edge:** ~10 MB

### Backup System

#### Auto-save
- All changes saved immediately
- No save button needed
- LocalStorage persistent

#### Manual Backup
- Export JSON anytime
- Timestamped files
- Restore on any device
